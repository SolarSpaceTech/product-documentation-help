# Combined Lists
This article provides an example of using ordered and unordered lists together.

<br/>

## Example of Usage

```md
Element types:
- Block:
  - Heading
    1. Level 1
      2. Level 2
      3. Level 3
      4. Level 4
      5. Level 5
      6. Level 6
  - Paragraph
  - Ordered and unordered lists
  - Divider
  - Indentation
  - Block code
  - Image
  - Information block
  - Table

- Inline
  - Link
  - Italic text
  - Bold text
  - Inline code

- Others:
  - HTML markup
```
